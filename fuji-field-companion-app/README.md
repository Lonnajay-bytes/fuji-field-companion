# Fuji Field Companion

A mobile-first progressive web app for the Fujifilm X-T5 and XF 16–80mm.

## Run locally

Because the app loads recipe data with JavaScript, open it through a local web server:

### Mac
1. Open Terminal.
2. `cd` into this folder.
3. Run: `python3 -m http.server 8000`
4. Open `http://localhost:8000`

### Install on iPhone
Host the folder on any static web host, open it in Safari, then use **Share → Add to Home Screen**.

## Included
- 120 original recipe starting points
- Recipe finder
- Search, filters, favorites, and comparison
- C1–C7 recommendation
- Learning center
- Exposure assistant
- Golden-hour planner
- Gear checklist
- Offline caching after first hosted load

## Notes
These are original starting points, not claimed reproductions of proprietary film stocks.
