FUJI FIELD COMPANION — TWO-BATCH GITHUB UPLOAD

UPLOAD BATCH 1 FIRST
1. Extract Fuji_Update_Batch_1.zip.
2. Open the extracted Fuji_Update_Batch_1 folder.
3. Press Command+A to select everything INSIDE it.
4. Drag the selection into your GitHub repository's Add file → Upload files screen.
5. Commit directly to main.

UPLOAD BATCH 2 SECOND
1. Extract Fuji_Update_Batch_2.zip.
2. Open the extracted Fuji_Update_Batch_2 folder.
3. Press Command+A to select everything INSIDE it.
4. Drag the selection into the same GitHub upload screen.
5. Commit directly to main.

Do not delete the existing root files first. GitHub will replace matching files.
The live GitHub Pages address stays the same.
