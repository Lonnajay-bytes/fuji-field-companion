Custom banks

Plan C1–C7 before touching the camera.

Your default set keeps five dependable recipes and leaves two rotating slots. Change any assignment and it saves on this device.

Enter a recipe in camera order

  1. IMAGE QUALITY SETTING → SELECT CUSTOM SETTING
  2. EDIT/SAVE CUSTOM SETTING → choose C1–C7
  3. Set Film Simulation and Dynamic Range
  4. Set White Balance and WB Shift
  5. Set Highlight, Shadow, Color, Sharpness, High ISO NR
  6. Set Grain, Color Chrome FX, Color Chrome FX Blue, Clarity
  7. Name the custom bank and save

XApp profile strategy

Build themed seven-bank groups such as Daily, Travel, Night, and Black Mist. Back up the entire camera configuration in XApp, then restore the profile that fits the shoot.

Important: restoring a profile changes the saved camera settings as a group; it is not a one-recipe “send to C6” button.
